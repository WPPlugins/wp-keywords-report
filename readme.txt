Know your blog position in Google SERP

== Description ==

**WP Keywords Report** Displays top keywords of your blog in Google SERP along with following data.

* Blog position in google for that keyword
* Blog URL in SERP for that keyword
* Keyword cost
* Percentage of google traffic from keyword
* Number of queries per day for that keyword
* Number of results in Google for that keyword

Online tool to [check keywords position](http://99webtools.com/organic-keywords-report.php)

== Installation ==

= Using the Plugin Manager =

1. Click Plugins
2. Click Add New
3. Search for `wp-keywords-report`
4. Click Install
5. Click Install Now
6. Click Activate Plugin

= Manually =

1. Upload `wp-keywords-report` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
